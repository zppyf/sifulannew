运行环境：phpnow  端口号：localhost

首页（index.html）:  
  1.鼠标滑过全部商品类目下的二级菜单出现三级菜单,鼠标从三级菜单上移出，三级菜单取消；
  2.右侧有楼梯点击到达相应栏目；
  3.右侧楼梯点击购物车可以跳转到购物车页面；
  4.护肤品牌楼层点击加入购物车可以将该商品加入购物车，点击商品图片进入对应详情页；
  5.点击注册可以跳转注册页面，注册成功会跳转登录页面，登录成功会跳转到首页；
  6.点击护肤品牌楼层的“全部”链接会跳转到商品列表页面。
  7.点击右上角购物车可以跳转到购物车页面。


详情页（lankouxiangqing.html）:
  1.点击左上角logo可以跳转到首页；
  2.鼠标滑过全部商品类目可以出现二级菜单，滑过二级菜单相应类目会出现对应三级菜单，离开相应三级菜单，二、三级菜单都消失；
  3.鼠标滑过左侧商品图片可出现放大镜效果，点击下面小图，可以切换上面的大图，显示对应的放大镜效果。


商品列表页（goods.html）:
  1.1.点击左上角logo可以跳转到首页；
  2.鼠标滑过全部商品类目可以出现二级菜单，滑过二级菜单相应类目会出现对应三级菜单，离开相应三级菜单，二、三级菜单都消失；
  3.点击热销排行里面的加入购物车可以将商品加入到购物车，点击商品图片会跳转到对应的商品详情页。


购物车页面（car.html）:
  1.点击左上角logo可以跳转到首页；
  2.可以实现商品的数量加减和删除功能。 


注册页面（zhuce.html）：
  1.数据接口：http://www.icodeilife.cn/ctrl/register.php；
  2.注册界面有表单验证，需要按规则注册账号，否则无法注册；
  3.点击右上角立即登录会跳转到登录页面；
  4.账号注册成功3秒后会自动跳转到登录页面。


登录页面（denglu.html）：
  1.数据接口：http://www.icodeilife.cn/ctrl/login.php；
  2.登录成功3秒后会自动跳转到首页；
  3.点击免费注册会跳转到注册页面。
